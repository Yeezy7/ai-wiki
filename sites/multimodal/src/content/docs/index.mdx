# 多模态

多模态学习（Multimodal Learning）致力于让模型同时理解和处理多种模态的信息（文本、图像、音频等）。

## 内容模块

<CardGrid>
  <Card title="CLIP" icon="star">
    对比学习、图文匹配、零样本分类
  </Card>
  <Card title="ViT" icon="magnifier">
    视觉 Transformer、图像分类、特征提取
  </Card>
  <Card title="Grounding" icon="puzzle">
    视觉定位、目标指代、区域描述
  </Card>
</CardGrid>

## 学习路径

1. **CLIP** — 理解对比学习和图文对齐
2. **ViT** — 掌握视觉 Transformer 架构
3. **Grounding** — 学习视觉定位技术
