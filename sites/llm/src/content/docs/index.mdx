# 大语言模型

大语言模型（Large Language Model, LLM）是自然语言处理领域的突破性技术，通过大规模预训练学习语言的深层规律。

## 学习路径

1. **Transformer 架构** — 理解自注意力机制和编码器-解码器结构
2. **注意力机制** — 深入理解各种注意力变体
3. **Fine-tuning** — 掌握模型微调技术
4. **RAG** — 学习检索增强生成技术
5. **RLHF** — 理解人类反馈强化学习
