# CV 面试题

## 深度学习基础

### 1. 什么是卷积神经网络（CNN）？

**标准回答：**

CNN 是一种专门用于处理网格结构数据（如图像）的深度神经网络。它的核心思想是通过卷积操作提取局部特征，并通过多层堆叠逐步提取更高层次的特征。

CNN 的三大核心组件：
- **卷积层**：通过卷积核滑动计算提取局部特征
- **池化层**：降低特征图尺寸，增加平移不变性
- **全连接层**：将特征映射到输出空间

CNN 的优势在于参数共享和局部连接，大大减少了参数量，同时保留了空间结构信息。

- 卷积操作的计算公式是什么？
- 什么是感受野？如何计算？
- 1x1 卷积的作用是什么？

### 2. BatchNorm 的作用和原理是什么？

**标准回答：**

BatchNorm（批归一化）用于解决深层网络训练中的内部协变量偏移问题。它在每个 mini-batch 上对特征进行归一化，使其均值为 0，方差为 1。

BatchNorm 的作用：
- 加速训练收敛
- 允许使用更大的学习率
- 具有正则化效果
- 缓解梯度消失/爆炸

训练和推理的区别：
- 训练时使用当前 batch 的统计量
- 推理时使用训练过程中累积的滑动平均统计量

## 目标检测

### 3. 一阶段和两阶段检测器的区别？

**标准回答：**

**两阶段检测器**（如 Faster R-CNN）：
- 第一阶段：生成候选区域（Region Proposals）
- 第二阶段：对候选区域进行分类和回归
- 精度高，但速度较慢

**一阶段检测器**（如 YOLO、SSD）：
- 直接在特征图上预测类别和位置
- 速度快，适合实时场景
- 精度相对较低，但差距在缩小

- Anchor-Based 和 Anchor-Free 的区别？
- YOLO 为什么这么快？
- DETR 为什么不需要 NMS？

### 4. NMS 的原理和流程？

**标准回答：**

NMS（非极大值抑制）用于去除高度重叠的冗余检测框，保留最可信的检测结果。

算法流程：
1. 按置信度从高到低排序所有候选框
2. 取出当前最高分框加入结果集
3. 计算该框与剩余所有框的 IoU
4. 删除 IoU 超过阈值的框
5. 重复步骤 2-4 直到没有候选框

NMS 的问题：
- 密集目标场景容易误删相邻目标
- 阈值选择影响结果

**变体：**
- Soft-NMS：降低重叠框的置信度而非直接删除
- DIoU-NMS：考虑中心点距离

### 5. mAP 是如何计算的？

**标准回答：**

mAP（mean Average Precision）是目标检测的核心评估指标。

计算步骤：
1. 计算每个类别的 Precision-Recall 曲线
2. 计算 AP（曲线下面积）
3. 对所有类别的 AP 求平均得到 mAP

AP 计算方式：
- 11 点插值法：在 0, 0.1, ..., 1.0 这 11 个点上取 Precision 的平均
- 所有点插值法：对所有 Recall 值对应的 Precision 求平均（COCO 使用）

## 图像分割

### 6. 语义分割、实例分割、全景分割的区别？

**标准回答：**

**语义分割（Semantic Segmentation）**：
- 对每个像素进行分类
- 不区分同一类别的不同实例
- 输出：像素级类别标签

**实例分割（Instance Segmentation）**：
- 检测目标并分割每个实例
- 区分同一类别的不同实例
- 输出：每个实例的掩码

**全景分割（Panoptic Segmentation）**：
- 结合语义分割和实例分割
- 对所有像素进行分类，同时区分实例
- 输出：像素级类别标签 + 实例 ID

**代表方法：**
- 语义分割：FCN、U-Net、DeepLab
- 实例分割：Mask R-CNN
- 全景分割：Panoptic FPN、Mask2Former
