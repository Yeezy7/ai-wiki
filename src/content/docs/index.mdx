# CV Wiki

CV Wiki 是一个面向计算机视觉学习、面试复习和工程实践的开源知识库。

它不是单纯的八股题库，而是希望把每个知识点讲成：

- **能学懂** — 从直觉出发，配合图示和代码
- **能面试** — 附带标准回答和高频追问
- **能落地** — 包含工程实践和常见坑
- **能查阅** — 结构化组织，快速定位

## 推荐学习路径

1. **深度学习基础** — 先打好基础，理解 CNN、BatchNorm、激活函数、损失函数
2. **图像处理基础** — 掌握滤波、边缘检测、形态学等经典方法
3. **目标检测** — 学习 R-CNN、YOLO 系列，理解 NMS、mAP 等核心概念
4. **图像分割** — 掌握语义分割、实例分割、全景分割的区别和方法
5. **模型部署** — 学习 ONNX、TensorRT、量化等工程化技术
6. **面试题库** — 按模块复习高频面试题
